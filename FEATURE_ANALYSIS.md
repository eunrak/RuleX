# NotebookLLM UI 분석 및 PolicyAI 추가 기능 제안

## 📊 NotebookLLM의 주요 UI/UX 특징

### 1. **3-Pane Layout (현재 구현됨 ✅)**
- 왼쪽: 소스 관리 (Sources)
- 중앙: 채팅 인터페이스 (Chat)
- 우측: 스튜디오/액션 패널 (Studio)

### 2. **소스 관리 패널 (왼쪽)**
#### 현재 구현된 기능:
- ✅ 정책서 목록 (Policies)
- ✅ 데이터베이스 연결 (Data Connections)
- ✅ 활성/비활성 상태 표시

#### 추가할 기능:
- 🆕 **소스 추가 버튼**: "소스 추가" 버튼을 더 눈에 띄게
- 🆕 **AI 추천 소스**: "Deep Research를 사용해 관련 정책서를 찾아보세요" 같은 프롬프트
- 🆕 **소스 검색**: "팀에서 새 소스를 검색하세요" 기능
- 🆕 **소스 필터링**: 언어, 날짜, 타입별 필터
- 🆕 **소스 미리보기**: 호버 시 문서 첫 페이지 미리보기

### 3. **채팅 인터페이스 (중앙)**
#### 현재 구현된 기능:
- ✅ AI 응답 및 사용자 메시지
- ✅ 액션 버튼 (View Violators, Create Rule 등)
- ✅ 데이터 테이블 표시

#### 추가할 기능:
- 🆕 **새로고침 버튼**: 채팅 영역 상단에 새로고침 아이콘
- 🆕 **제안된 질문 (Suggested Prompts)**: 
  - "지난 주 정책 위반 사항은?"
  - "가장 많이 위반되는 규칙은?"
  - "신규 직원을 위한 보안 정책 요약해줘"
- 🆕 **소스 인용 표시**: AI 응답에 어떤 정책서의 몇 페이지를 참조했는지 표시
  - 예: "IT Security Standard v2.0, Section 4.1 참조"
- 🆕 **응답 저장/북마크**: 중요한 AI 응답을 저장
- 🆕 **응답 공유**: 팀원에게 공유할 수 있는 링크 생성

### 4. **스튜디오 패널 (우측)**
#### 현재 구현된 기능:
- ✅ Active Violations (위반 사항 모니터링)
- ✅ Active Rules (활성 규칙 목록)

#### 추가할 기능:
- 🆕 **AI 오디오 개요**: 정책서를 음성으로 요약 (TTS)
- 🆕 **타임라인**: 정책 변경 이력 시각화
- 🆕 **FAQ 자동 생성**: 정책서 기반 FAQ 자동 생성
- 🆕 **브리핑 문서**: 경영진용 요약 보고서 자동 생성
- 🆕 **퀴즈 생성**: 직원 교육용 퀴즈 자동 생성
- 🆕 **학습 가이드**: 신입 직원 온보딩 자료 생성
- 🆕 **메모 추가**: 사용자가 직접 인사이트 기록
- 🆕 **인용 및 통찰력**: 정책서에서 중요한 부분 하이라이트

### 5. **상단 헤더**
#### 추가할 기능:
- 🆕 **공유 버튼**: 현재 대화를 팀원과 공유
- 🆕 **설정 버튼**: 모델 선택, 테마 변경 등
- 🆕 **앱 메뉴**: 다른 기능으로 빠르게 이동

## 🎯 우선순위별 구현 계획

### Phase 1: 즉시 적용 가능한 기능 (데모용)
1. **제안된 질문 (Suggested Prompts)** - 사용자 경험 개선
2. **소스 인용 표시** - AI 신뢰도 향상
3. **새로고침 버튼** - 채팅 재생성 기능
4. **메모 추가** - 우측 패널에 간단한 메모 기능

### Phase 2: 중요 기능 (1-2주 내)
5. **FAQ 자동 생성** - 정책서 기반 FAQ 생성
6. **브리핑 문서 생성** - 요약 보고서 자동 생성
7. **타임라인** - 정책 변경 이력 시각화
8. **소스 추천** - AI 기반 관련 문서 추천

### Phase 3: 고급 기능 (장기)
9. **AI 오디오 개요** - TTS 통합
10. **퀴즈 생성** - 교육용 콘텐츠 자동 생성
11. **학습 가이드** - 온보딩 자료 자동 생성

## 💡 UI/UX 개선 사항

### 디자인 측면:
1. **다크 모드**: NotebookLLM은 다크 테마 사용 - PolicyAI도 다크 모드 추가
2. **아이콘 일관성**: 모든 기능에 직관적인 아이콘 사용
3. **카드 기반 레이아웃**: 스튜디오 패널의 각 기능을 카드로 표시
4. **애니메이션**: 부드러운 전환 효과 (이미 Framer Motion 사용 중 ✅)

### 인터랙션 측면:
1. **드래그 앤 드롭**: 소스 업로드 시 드래그 앤 드롭 지원
2. **키보드 단축키**: 빠른 작업을 위한 단축키 (Cmd+K로 검색 등)
3. **실시간 피드백**: 로딩 상태, 진행률 표시
4. **에러 핸들링**: 친절한 에러 메시지
